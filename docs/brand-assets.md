# GitRescue Brand Assets

## Core Idea

GitRescue combines two visual ideas:

- Git branch nodes: the product understands repository state.
- Rescue shield: the product makes risky moments safer.

The mark should feel practical, trustworthy, and developer-native. Avoid cute
mascots or overly glossy AI-tool styling.

## Palette

- Git orange: `#F05033`
- Deep ink: `#171717`
- Soft stone: `#FAFAF9`
- Muted text: `#6B7280`
- Shadow red: `#A12E1E`

## Assets

- `media/icon.png`: 128x128 Marketplace icon, generated by `npm run gen:icon`.
- `media/gitrescue-icon.svg`: vector icon source.
- `media/gitrescue-logo-horizontal.svg`: horizontal logo and wordmark.
- `media/gitrescue-marketplace-banner.svg`: wide Marketplace/README hero.
- `media/gitrescue-product-hunt-card.svg`: Product Hunt/social launch card.
- `media/png/`: exact-size PNG exports for upload surfaces.

## Usage

Use `media/icon.png` for the VS Code Marketplace `icon` field.

Use the horizontal SVG for GitHub README and launch decks. Use the banner/card
SVGs for launch imagery and export PNGs from them when a platform requires
bitmap uploads.

Current PNG exports:

- `media/png/gitrescue-icon.png`: 128x128
- `media/png/gitrescue-logo-horizontal.png`: 760x180
- `media/png/gitrescue-marketplace-banner.png`: 1280x640
- `media/png/gitrescue-product-hunt-card.png`: 1270x760
