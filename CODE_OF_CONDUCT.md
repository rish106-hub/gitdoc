# Code of Conduct

## Our Pledge

We want GitRescue to be a welcoming project for developers at every Git skill
level. Be kind, direct, and patient.

## Expected Behavior

- Assume good intent.
- Explain technical feedback clearly.
- Be respectful of beginners.
- Keep criticism focused on code, design, safety, and product behavior.
- Avoid harassment, insults, or personal attacks.

## Unacceptable Behavior

- Harassment or discriminatory language.
- Publicly sharing private information.
- Dismissing beginners for not knowing Git internals.
- Pressuring maintainers to merge unsafe changes.

## Enforcement

Maintainers may edit, hide, or remove comments and may block contributors who
violate this code of conduct.
