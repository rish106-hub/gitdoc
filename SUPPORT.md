## Install and Marketplace Questions

Use the VS Code Marketplace listing:

https://marketplace.visualstudio.com/items?itemName=rish106-hub.git-rescue

## Before Filing a Bug

Please include:

- VS Code or Cursor version
- GitRescue version
- operating system
- the Git state or pasted Git error
- whether this happened in a real repo or a throwaway demo repo

Do not include private repository URLs, secrets, or tokens.
